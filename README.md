<h2>Capstone Project</h2>

This was a semester-long group project where we were tasked to create a 2D fighting game.  My contribution
consists of coding the UI, some gameplay traps, and the win-lose scenes.  I was also in charge of most of the art, 
including the UI art and character animations.
